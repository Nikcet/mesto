# Учебный проект

### Описание

Проект для обучения.
Все кликается. Можно добавлять карточки, удалять, лайкать, редактировать информацию.
Имеется валидация данных при заполнении информации.
Вся информация подтягивается с сервера.


### Технологии

* HTML
* CSS
* JS (vanilla)
* Figma
